# Expense_Tracker
Expense Tracker (Java)

A simple console-based Expense Tracker application built in Java.
This project allows users to record, manage, and analyze daily expenses using a menu-driven interface. It demonstrates core Java concepts such as Object-Oriented Programming, Collections, and File Handling.

📌 Features

➕ Add new expenses

📋 View all recorded expenses

🔍 Filter expenses by category

📊 Calculate total spending

💾 Save expenses to file

📅 Track date for each expense

🛠 Technologies Used

Java

Java Collections (ArrayList)

File Handling (ObjectInputStream / ObjectOutputStream)

OOP Concepts (Classes, Objects, Encapsulation)

📂 Project Structure
ExpenseTrackerAdvanced/
│
├── Expense.java          # Expense model class
├── ExpenseManager.java   # Business logic for managing expenses
├── FileManager.java      # Handles saving/loading data from file
└── Main.java             # Main program with menu interface
⚙️ How to Compile and Run
1. Clone the Repository
git clone https://github.com/yourusername/expense-tracker-java.git
2. Navigate to the Project Folder
cd expense-tracker-java
3. Compile the Java Files
javac *.java
4. Run the Application
java Main

📦 Data Storage

All expenses are stored in a file:

This allows the application to retain data even after closing the program.
